# `docs/**/entities/*-entity.md` - Entityの定義

Entity（or 集約）を定義。

- 属性には制約を含める
- ビジネスルールは明確かつ検証可能な形で記述する
- 他の値オブジェクトやEntityを使用する
- テーブルを使用しない

```
# [モデル名]

[モデルの役割と目的の説明]

## 属性

### [属性名A]

[属性の役割と目的の説明]

- ビジネスルール

### [属性名B]

## ビジネスルール

その他のビジネスルールをここに記述してください。

- [ルール1]
- [ルール2]
```

必要に応じてユーザに提案と共に質問して詳細を引き出してください。
